HTML

LISTA DE ETIQUETA DE HTML
ETIQUETA DESCRPCION
html:5 referencia del software
h1 al h6 encabezado implementan seis niveles
link vínculo con otro recurso.
p distribuir el texto en párrafos.
body representa el contenido de un documento HTML
head provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos
meta para aportar información sobre el documento.
title le permite darle un título a una página web
class lista de las clases del elemento separada por espacios .
id identificar el elemento al vincularlo
img representa una imagen en el documento.
span para aplicar estilo al texto o agrupar elementos en línea.
div definir un bloque de contenido o sección de la página, para poder aplicarle diferentes estilos e incluso para realizar operaciones sobre ese bloque específico.
a crea un enlace a otras páginas de internet, archivos o ubicaciones dentro de la misma página, direcciones de correo, o cualquier otra URL.
hr es un elemento utilizado para crear una línea horizontal en un documento web
br inserta un quiebre de línea, causando que el texto continúe en una línea nueva.
table presentar datos de tabla y establecer el diseño de texto y gráficos en una página HTML.
tr define una fila de celdas en una tabla.
th inserta una celda de encabezado en una tabla
td define la celda de una tabla que contiene datos
caption darle un título descriptivo a las tablas.
thead debe delimitar el encabezado de la tabla;
tfoot define una sección de pie en una tabla
button crea botones marcadores.
